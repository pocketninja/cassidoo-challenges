Given an int array coins and an int amount, return an array of coins that add up to amount (and an empty array if it’s an impossible combination).

Example:

let coins = [2,3,5,7]
let amount = 17

$ coinCombo(coins, amount)
$ [3,7,7]

$ coinCombo([2,3,5], 11)
$ [3,3,5]
